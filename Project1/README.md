# INTD 555 Project 1: Regression Analysis

This project analyzes the relationship between advertising expenditures and sales using TV, Radio, and Social Media promotion data. Linear regression is used to evaluate which marketing channels are the strongest predictors of sales.

## Dataset
The dataset includes:
- TV promotion budget (in millions)
- Radio promotion budget (in millions)
- Social Media promotion budget (in millions)
- Influencer type (Mega, Macro, Micro, Nano)
- Sales (in millions)

Source: Saragih, H. (n.d.). Dummy advertising and sales data. Kaggle. https://www.kaggle.com/datasets/harrimansaragih/dummy-advertising-and-sales-data/

## Project Files
- `INTD555_Project1_Regression_Analysis.ipynb` – main analysis notebook
- `Dummy Data HSS.csv` – dataset used for analysis

## Methods
- Data cleaning and preprocessing
- Exploratory data analysis (scatter plots)
- Correlation analysis (Pearson, Spearman, Kendall)
- Linear regression modeling

## Key Findings
- TV promotion is the strongest predictor of sales
- Radio promotion shows a strong but more variable relationship
- Social Media promotion has a weaker relationship and higher variability
