# Acoustic Fire Extinguisher – Binary Classification Project

## Overview
This project builds a binary classification model to predict whether a flame will be extinguished using acoustic signals. The dataset includes physical and acoustic measurements collected during controlled fire extinguishing experiments.

The goal is to analyze the dataset, build a baseline model, and improve its performance using feature selection and hyperparameter tuning.

---

## Dataset
The dataset used in this project is the **Acoustic Fire Extinguisher Dataset**, which contains 17,442 observations from experiments involving different fuel types and sound wave conditions.

### Features:
- SIZE – Flame size (encoded numerically)
- FUEL – Type of fuel (gasoline, kerosene, thinner, LPG)
- DISTANCE – Distance from sound source (cm)
- DESIBEL – Sound intensity (dB)
- AIRFLOW – Airflow generated by sound waves (m/s)
- FREQUENCY – Sound frequency (Hz)

### Target:
- STATUS – Flame extinguished (1) or not (0)

The dataset is well-balanced, making it suitable for binary classification.

## Methodology

### 1. Exploratory Data Analysis (EDA)
- Visualized relationships using boxplots, scatter plots, and count plots  
- Identified key predictors such as airflow and decibel level  

### 2. Baseline Model
- Logistic Regression was used as a baseline classifier  
- Evaluated using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score  

### 3. Model Improvement
- Feature selection using Random Forest feature importance  
- Hyperparameter tuning using GridSearchCV  
- Alternative model: Random Forest Classifier  

---

## Results

| Model | Accuracy | F1 Score |
|------|--------|--------|
| Logistic Regression | ~0.91 | ~0.91 |
| Random Forest | ~0.94 | ~0.94 |

The Random Forest model outperformed the baseline, demonstrating improved ability to capture non-linear relationships.
